# MvzickPlayer <Badge type="danger">archived</Badge>
MvzickPlayer是一款基于Unity开发的本地音乐播放器应用。

::: warning 说明
此项目已被存档
:::

## 关于项目

- 通过[taglib-sharp](https://github.com/mono/taglib-sharp.git)读取音频元数据
- 通过UGUI构建用户界面
  - 构建了较完善的UI管理框架
  - 对UGUI原生组件封装，构建了模态对话框系统、虚拟列表、共享元素过渡等功能
- 采用分层架构，业务逻辑完全解耦Unity。
